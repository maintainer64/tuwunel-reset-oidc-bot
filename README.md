# Tuwunel Reset OIDC Bot

Matrix-бот для сброса паролей пользователей через админскую комнату.

## Возможности

- Обычный Matrix пользователь
- Поддержка E2EE (сквозного шифрования)
- Автоматическое сохранение сессии в SQLite
- При запуске устанавливает display name и аватар
- Работает через sync (long polling)
- Пользователь пишет боту в личку → бот предлагает сбросить пароль
- Пользователь отвечает "сброс" → бот генерирует временный пароль
- Бот отправляет команду `!admin users reset-password` в админскую комнату
- Бот отправляет временный пароль пользователю в личку
- Через 5 минут бот удаляет сообщение с паролем и меняет пароль на случайный

## Конфигурация

Создайте файл `.env`:

```bash
# Homeserver URL
MATRIX_HOMESERVER=https://matrix.yourserver.com

# Username бота (localpart)
MATRIX_BOT_USERNAME=reset-bot

# Пароль бота (только для первого запуска)
MATRIX_BOT_PASSWORD=your_bot_password

# Ключ для шифрования E2EE (обязательно измените!)
MATRIX_PICKLE_KEY=your-secret-pickle-key-change-me

# Опционально: display name бота
MATRIX_BOT_DISPLAYNAME=Сброс пароля

# Опционально: URL аватара (mxc://...)
MATRIX_BOT_AVATAR=mxc://...

# Admin комната для отправки команд
ADMIN_ROOM_ID=!admin_room_id:matrix.yourserver.com
```

## Запуск

### Первый запуск

```bash
# Запустите с паролем - бот залогинится и сохранит всё в SQLite
MATRIX_BOT_PASSWORD=your_password go run .
```

### Последующие запуски

```bash
# Просто запустите - всё загрузится из SQLite
go run .
```

### Docker

```bash
# 1. Создайте .env с паролем и pickle key

# 2. Запустите
docker-compose up -d

# 3. Удалите пароль из .env (он больше не нужен)
# Теперь всё хранится в ./data/
docker-compose restart
```

## Требования для сборки

Для сборки с E2EE требуется libolm:

```bash
# macOS
brew install libolm

# Alpine (Docker)
apk add libolm-dev
```

## Использование

1. Создайте пользователя для бота (через Element или admin API)
2. Пригласите бота в личку
3. Напишите любое сообщение
4. Бот предложит ответить "сброс" или "reset"
5. После ответа получите временный пароль на 5 минут

## Как это работает

1. Бот - обычный Matrix пользователь
2. При первом запуске с паролем - логинится и сохраняет всё в SQLite
3. Криптографические ключи и токены хранятся в базе (`crypto.db`)
4. При последующих запусках - просто загружает всё из базы
5. Пароль больше не нужен после первого запуска
